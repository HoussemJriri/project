1. how does the web works ? 
Computers connected to the web are called clients and servers.
The client requests the server with a language called HTTP "HyperText Transfer Protocol". This is why web page addresses begin with http:// 
The server respond by sending pages when they are requested by a web browser, such as Google Chrome.
